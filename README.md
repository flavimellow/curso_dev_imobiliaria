# 🏠 Projeto Dev Imobiliária

Este projeto é um site institucional fictício de uma imobiliária, desenvolvido durante estudos de **HTML e CSS** para praticar a criação de páginas web estruturadas e estilizadas.

O objetivo do projeto é simular um site real de uma empresa do setor imobiliário, apresentando imóveis, informações institucionais e páginas de contato.

---

# 🚀 Tecnologias utilizadas

- HTML5
- CSS3
